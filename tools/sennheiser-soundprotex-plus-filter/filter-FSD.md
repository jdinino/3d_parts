# Total Block Filter Full Specifications Document

## Overview

Replacement total block filter cap for Sennheiser SoundProtex Plus reusable earplugs. Provides complete sound isolation when full hearing protection is needed.

## Filter Specifications

| Parameter | Value |
|-----------|-------|
| Total Height | 4.5mm |
| Base Diameter | 9mm |
| Stem Diameter | 6mm |
| Hole Diameter | 4mm |
| Base Thickness | 0.9mm |
| Stem Height | 3.6mm |
| Groove Height | 1.5mm |
| Groove Depth | 0.5mm |
| Material | TPU, PETG, PLA |

### Design Features
- Wide base for sealing against earplug housing
- Narrow stem with grip groove for removal
- Solid base blocks all sound transmission
- Draft angle on hole for printability
- Bottom chamfer for bed adhesion

## Material Requirements

| Material | Status | Reason |
|----------|--------|--------|
